# PROYECTO 2 - Gestión de almacén y material fungible
<b>Contexto</b></br>
Una empresa industrial pierde dinero porque <b>no sabe exactamente que material tiene</b>, ni cuándo reponerlo.</br>
<b>Objetivo</b></br>
Digitalizar el control de stock de un almacén.</br>
<b>Funcionalidades mínimas</b></br>
<ul>
  <li>Alta de productos (nombre,categoría,proveedor).</li>
  <li>Cantidad disponible.</li>
  <li>Entradas y salidas de material.</li>
  <li>Aviso de stock mínimo.</li>
</ul>
<b>Ampliaciones</b></br>
<ul>
  <li>Gráficas de consumo mensual</li>
  <li>Historial de movimientos.</li>
  <li>Usuarios por departamento.</li>
</ul>
<b>Imagen de referencia</b></br>
